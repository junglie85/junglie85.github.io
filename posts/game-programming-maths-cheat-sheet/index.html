Game Programming Maths Cheat Sheet

26 Jan 2022
1 minute read
games , maths

I don’t use these maths functions often enough for them to stick in my head when I need them. I’m developing this cheat sheet to help me have a handy reference when needed.

Linear interpolation

Linear interpolation or LERP is useful for finding the value of a number between two points. Examples of uses are colour blending and distance along a line.

There are two formulae:

c = a + (b - a) * t

c = a * (1 - t) + b * t

Where a is the starting point, b is the end point and t is the distance as a percentage between them.
